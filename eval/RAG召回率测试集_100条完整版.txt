# RAG召回率评估测试集（企业知识库场景）
# 说明：本测试集共100条，覆盖6大常见企业知识库场景，包含6类问题类型，可直接用于RAG检索效果评估
# 格式说明：每条数据包含【测试编号、问题、正确答案片段、问题类型、所属分类】

====================
测试编号：001
问题：员工工作满3年可以享受多少天年假？
正确答案片段：员工工作满1年不满10年的，年休假5天；已满10年不满20年的，年休假10天；已满20年的，年休假15天。国家法定休假日、休息日不计入年休假的假期。
问题类型：事实类
所属分类：考勤与休假制度
====================
测试编号：002
问题：年假可以和法定节假日连休吗？
正确答案片段：年休假可以与法定节假日、休息日连休，员工需提前7个工作日提交休假申请，经直属领导审批通过后方可休假。
问题类型：事实类
所属分类：考勤与休假制度
====================
测试编号：003
问题：试用期员工能不能休年假？
正确答案片段：试用期员工累计工作满1年的，可享受年休假，试用期计入员工工作年限。试用期未满1年的，不享受当年年假。
问题类型：细节类
所属分类：考勤与休假制度
====================
测试编号：004
问题：当年没休完的年假怎么处理？
正确答案片段：员工当年未休完的年假，最多可结转至次年3月31日前使用，逾期未使用的自动作废，公司不予折现补偿。员工因工作原因无法休年假的，经部门负责人审批后可按日工资的300%支付未休年假工资。
问题类型：总结类
所属分类：考勤与休假制度
====================
测试编号：005
问题：请病假需要提供什么材料？
正确答案片段：员工请病假1天以内的，需提前告知直属领导；病假3天及以上的，需提供二级及以上医院开具的诊断证明和病假条，提交至HR部门备案。病假期间工资按基本工资的80%发放。
问题类型：事实类
所属分类：考勤与休假制度
====================
测试编号：006
问题：事假最多可以连续请多少天？
正确答案片段：员工单次事假不得超过5天，年度累计事假不得超过15天。连续事假超过3天的，需经部门总监审批。事假期间扣除当日全额工资。
问题类型：事实类
所属分类：考勤与休假制度
====================
测试编号：007
问题：婚假有多少天？再婚可以休婚假吗？
正确答案片段：员工依法办理结婚登记的，可享受10天婚假，再婚员工与初婚员工享受同等婚假待遇。婚假需在领取结婚证后1年内一次性休完，不得拆分。
问题类型：对比类
所属分类：考勤与休假制度
====================
测试编号：008
问题：产假有多少天？难产和多胞胎怎么算？
正确答案片段：女职工生育享受98天产假，其中产前可以休假15天；难产的，增加产假15天；生育多胞胎的，每多生育1个婴儿，增加产假15天。符合地方政策的额外增加的产假按当地规定执行。
问题类型：细节类
所属分类：考勤与休假制度
====================
测试编号：009
问题：迟到多久算旷工？
正确答案片段：员工迟到或早退30分钟以内的，扣除当月全勤奖的20%；迟到30分钟以上2小时以内的，按事假半天处理；迟到2小时以上的，按旷工1天处理。月度累计旷工3天及以上的，按严重违纪处理。
问题类型：事实类
所属分类：考勤与休假制度
====================
测试编号：010
问题：全勤奖的发放条件是什么？
正确答案片段：月度内无迟到、早退、旷工、事假、病假的员工，可获得全额全勤奖。月度内有1次迟到10分钟以内的，扣除50%全勤奖；有2次及以上迟到或任何早退、事假、病假的，扣除当月全部全勤奖。
问题类型：总结类
所属分类：考勤与休假制度
====================
测试编号：011
问题：工作满10年的员工，年假和工作满5年的有什么区别？
正确答案片段：员工工作满1年不满10年的，年休假5天；已满10年不满20年的，年休假10天；已满20年的，年休假15天。
问题类型：对比类
所属分类：考勤与休假制度
====================
测试编号：012
问题：陪产假有多少天？
正确答案片段：男职工配偶生育的，可享受15天陪产假，陪产假需在配偶生育前后1个月内休完，陪产假期间工资全额发放。
问题类型：事实类
所属分类：考勤与休假制度
====================
测试编号：013
问题：丧假的适用范围和天数是多少？
正确答案片段：员工直系亲属（父母、配偶、子女）去世的，可享受3天丧假；祖父母、外祖父母、岳父母、公婆去世的，可享受1天丧假。丧假期间工资全额发放。
问题类型：事实类
所属分类：考勤与休假制度
====================
测试编号：014
问题：年假的申请流程是什么？
正确答案片段：员工申请年假需提前3个工作日在OA系统提交休假申请，1天以内的年假由直属领导审批；3天及以上的年假需经部门负责人审批。年假申请审批通过后方可休假，未经审批擅自休假按旷工处理。
问题类型：流程类
所属分类：考勤与休假制度
====================
测试编号：015
问题：员工离职的时候，未休的年假怎么算？
正确答案片段：员工离职时，当年未休的年假，按照员工当年已工作时间折算应休未休年休假天数，折算后不足1整天的部分不享受年休假。折算方法：（当年度在本单位已过日历天数÷365天）×职工本人全年应当享受的年休假天数-当年度已安排年休假天数。未休年假按日工资的300%支付补偿。
问题类型：细节类
所属分类：考勤与休假制度
====================
测试编号：016
问题：旷工1天怎么扣工资？
正确答案片段：旷工1天扣除当日全额工资，同时扣除当月全部全勤奖。月度累计旷工2天的，给予书面警告；累计旷工3天及以上的，按严重违纪处理，公司有权解除劳动合同。
问题类型：事实类
所属分类：考勤与休假制度
====================
测试编号：017
问题：加班可以调休吗？调休有效期是多久？
正确答案片段：工作日加班和周末加班可以申请调休，法定节假日加班不得调休，按3倍工资发放。调休有效期为6个月，逾期未使用的调休自动作废，不予折现。
问题类型：事实类
所属分类：考勤与休假制度
====================
测试编号：018
问题：病假超过多久会影响年终奖？
正确答案片段：年度累计病假超过30天的，扣除20%年终奖；超过60天的，扣除50%年终奖；超过90天的，不享受当年年终奖。工伤病假不影响年终奖发放。
问题类型：细节类
所属分类：考勤与休假制度
====================
测试编号：019
问题：事假和病假的工资发放标准有什么区别？
正确答案片段：事假期间扣除当日全额工资；病假期间工资按基本工资的80%发放，病假工资不得低于当地最低工资标准的80%。
问题类型：对比类
所属分类：考勤与休假制度
====================
测试编号：020
问题：新入职员工当年的年假怎么算？
正确答案片段：新入职员工连续工作满12个月以上的，当年度年休假天数，按照在本单位剩余日历天数折算确定，折算后不足1整天的部分不享受年休假。折算方法：（当年度在本单位剩余日历天数÷365天）×职工本人全年应当享受的年休假天数。
问题类型：细节类
所属分类：考勤与休假制度
====================
测试编号：021
问题：工资每个月什么时候发？
正确答案片段：公司每月15日发放上月工资，如遇节假日或休息日，则提前到最近的工作日发放。工资通过银行转账方式发放至员工本人银行卡。
问题类型：事实类
所属分类：薪酬福利制度
====================
测试编号：022
问题：五险一金的缴纳基数是怎么定的？
正确答案片段：五险一金缴纳基数按照员工上年度月平均工资确定，缴费基数上限为当地社平工资的300%，下限为当地社平工资的60%。新入职员工按照首月约定工资作为缴费基数。
问题类型：事实类
所属分类：薪酬福利制度
====================
测试编号：023
问题：年终奖的发放时间和条件是什么？
正确答案片段：年终奖每年1月发放上一年度年终奖，发放当日仍在职的员工可享受全额年终奖。年度绩效考核不合格、年度累计旷工3天及以上、当年离职的员工不享受年终奖。
问题类型：总结类
所属分类：薪酬福利制度
====================
测试编号：024
问题：公司有哪些补贴？
正确答案片段：公司为员工提供以下补贴：1. 餐补：工作日每天20元，随当月工资发放；2. 交通补贴：每月300元；3. 通讯补贴：每月100元；4. 住房补贴：异地入职员工可享受每月1000元住房补贴，补贴期限1年。
问题类型：总结类
所属分类：薪酬福利制度
====================
测试编号：025
问题：试用期工资是正式工资的多少？
正确答案片段：员工试用期工资不得低于劳动合同约定工资的80%，且不得低于当地最低工资标准。试用期工资和正式工资同样缴纳五险一金。
问题类型：事实类
所属分类：薪酬福利制度
====================
测试编号：026
问题：加班费的计算标准是什么？
正确答案片段：加班费计算标准：1. 工作日加班：按小时工资的150%计算；2. 周末加班：按日工资的200%计算或安排调休；3. 法定节假日加班：按日工资的300%计算，不得调休。小时工资=月工资÷21.75÷8。
问题类型：事实类
所属分类：薪酬福利制度
====================
测试编号：027
问题：绩效工资占工资的比例是多少？
正确答案片段：员工工资结构为：基本工资（60%）+ 绩效工资（30%）+ 补贴（10%）。绩效工资根据月度绩效考核结果发放，绩效考核S级发放120%绩效工资，A级发放100%，B级发放80%，C级发放50%，D级不发放绩效工资。
问题类型：细节类
所属分类：薪酬福利制度
====================
测试编号：028
问题：年度调薪的时间和规则是什么？
正确答案片段：公司每年4月统一进行年度调薪，调薪幅度根据员工上年度绩效考核结果确定：S级调薪15%-20%，A级调薪8%-12%，B级调薪3%-5%，C级及以下不调薪。连续两年绩效考核S级的员工可额外获得晋升调薪机会。
问题类型：总结类
所属分类：薪酬福利制度
====================
测试编号：029
问题：个税是公司代扣代缴吗？
正确答案片段：员工工资薪金所得的个人所得税由公司统一代扣代缴，公司会在每月工资条中列明个税扣除金额，员工可在个人所得税APP查询纳税记录。
问题类型：事实类
所属分类：薪酬福利制度
====================
测试编号：030
问题：离职员工的工资什么时候结算？
正确答案片段：员工离职当日，公司一次性结清员工全部工资、加班费、未休年假补偿等所有应付款项。离职当月五险一金缴纳至员工离职当月。
问题类型：事实类
所属分类：薪酬福利制度
====================
测试编号：031
问题：员工可以申请预支工资吗？
正确答案片段：员工入职满3个月，可申请预支不超过当月已工作天数的工资，每月最多预支1次，预支工资从当月工资中扣除。预支工资需提前3个工作日提交申请，经部门负责人和HR审批通过后发放。
问题类型：事实类
所属分类：薪酬福利制度
====================
测试编号：032
问题：年终奖要扣个税吗？
正确答案片段：年终奖需缴纳个人所得税，居民个人取得全年一次性奖金，在2027年12月31日前，可选择不并入当年综合所得，单独计算纳税，也可选择并入当年综合所得计算纳税。
问题类型：事实类
所属分类：薪酬福利制度
====================
测试编号：033
问题：公司有员工体检福利吗？
正确答案片段：公司每年为正式员工提供一次免费体检，体检标准为每人500元，体检时间为每年5-6月，员工可选择公司合作体检机构或自行体检后凭发票报销。试用期员工不享受年度体检福利。
问题类型：事实类
所属分类：薪酬福利制度
====================
测试编号：034
问题：餐补是只要上班就有吗？
正确答案片段：工作日正常出勤的员工享受当日餐补，请假（事假、病假、年假）、旷工、出差的员工不享受当日餐补。餐补随当月工资发放，无需员工提交发票。
问题类型：细节类
所属分类：薪酬福利制度
====================
测试编号：035
问题：住房补贴的申请条件是什么？
正确答案片段：异地入职员工（户口和常住地均不在工作城市），入职后可申请每月1000元住房补贴，补贴期限为12个月，补贴随当月工资发放。员工在工作城市购房后，不再享受住房补贴。
问题类型：事实类
所属分类：薪酬福利制度
====================
测试编号：036
问题：加班餐补和交通补怎么申请？
正确答案片段：员工工作日加班至20:00以后的，可享受50元加班餐补；加班至22:00以后的，可报销打车费用。加班餐补和打车费需在OA系统提交申请，附加班记录和发票，经审批后报销。
问题类型：流程类
所属分类：薪酬福利制度
====================
测试编号：037
问题：基本工资和绩效工资有什么区别？
正确答案片段：基本工资是固定发放的部分，占工资总额的60%，不受绩效考核影响；绩效工资是浮动部分，占工资总额的30%，根据月度绩效考核结果发放，考核结果不同发放比例不同。
问题类型：对比类
所属分类：薪酬福利制度
====================
测试编号：038
问题：婚假、产假期间工资怎么发？
正确答案片段：婚假、产假、陪产假、丧假期间，员工工资全额发放，不影响绩效工资和全勤奖。病假期间工资按基本工资的80%发放，事假期间扣除全额工资。
问题类型：总结类
所属分类：薪酬福利制度
====================
测试编号：039
问题：年度奖金和年终奖是一回事吗？
正确答案片段：年度奖金包含年终奖和特别贡献奖，年终奖是全员普发的年度奖励，特别贡献奖是授予年度有突出贡献的员工的额外奖励，金额为1-3个月工资，不与年终奖冲突。
问题类型：对比类
所属分类：薪酬福利制度
====================
测试编号：040
问题：社保断缴有什么影响？
正确答案片段：社保断缴会影响：1. 医疗保险：断缴次月无法享受医保待遇；2. 养老保险：累计缴费年限不受影响，但会影响退休后养老金金额；3. 购房、购车、落户资格：多数城市要求社保连续缴纳一定年限，断缴会重新计算连续缴纳时间。
问题类型：总结类
所属分类：薪酬福利制度
====================
测试编号：041
问题：新员工入职需要准备什么材料？
正确答案片段：新员工入职需准备以下材料：1. 身份证原件及复印件3份；2. 学历、学位证书原件及复印件；3. 离职证明原件；4. 1寸白底照片3张；5. 银行卡复印件；6. 社保公积金转移材料。
问题类型：总结类
所属分类：入职离职流程
====================
测试编号：042
问题：试用期最长是多久？
正确答案片段：劳动合同期限3个月以上不满1年的，试用期不得超过1个月；劳动合同期限1年以上不满3年的，试用期不得超过2个月；3年以上固定期限和无固定期限的劳动合同，试用期不得超过6个月。同一用人单位与同一劳动者只能约定一次试用期。
问题类型：事实类
所属分类：入职离职流程
====================
测试编号：043
问题：转正申请什么时候提交？
正确答案片段：员工试用期结束前7个工作日，可在OA系统提交转正申请，附试用期工作总结，经直属领导、部门负责人、HR审批通过后，正式转正。试用期考核不合格的，可延长试用期或解除劳动合同。
问题类型：流程类
所属分类：入职离职流程
====================
